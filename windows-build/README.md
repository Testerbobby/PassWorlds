# Windows Build

Готовая Windows версия генератора паролей (без установки Python).

## Скачать

Перейдите в раздел [Releases](https://github.com/Testerbobby/PassWorlds/releases) для скачивания готовых .exe файлов.

## Сборка из исходников

1. Установите Python 3.x
2. Скачайте репозиторий
3. Запустите `build_windows.bat`

После сборки исполняемые файлы будут в папке `dist/`:
- `PassWorlds_GUI.exe` — графическая версия
- `PassWorlds_CLI.exe` — консольная версия

# PassWorlds

Генератор безопасных паролей на Python с CLI и GUI интерфейсами.

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

## Возможности

- Генерация криптографически стойких паролей
- Настройка длины пароля (4-64 символа в GUI, 4-128 в CLI)
- Выбор типов символов: заглавные, строчные буквы, цифры, специальные символы
- Оценка надёжности пароля
- **Проверка на утечки** (Have I Been Pwned API)
- **История сгенерированных паролей**
- **Расчёт энтропии** пароля
- **Экспорт/импорт настроек**
- **Смена темы** (тёмная/светлая/системная)
- Копирование в буфер обмена
- Два интерфейса: командная строка (CLI) и графический (GUI)

## Сборка из исходников

### Требования

- Python 3.8+
- pip

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск из исходников

#### GUI версия
```bash
python password_generator_ui.py
```

#### CLI версия
```bash
python password_generator.py
```

### Горячие клавиши (GUI)

| Комбинация | Действие |
|------------|----------|
| `Ctrl+G` | Сгенерировать пароль |
| `Ctrl+C` | Копировать пароль |
| `Ctrl+H` | Открыть историю |

### Параметры CLI

```bash
python password_generator.py [опции]

Опции:
  -l, --length N        Длина пароля (по умолчанию: 16)
  -u, --no-upper        Без заглавных букв
  -L, --no-lower        Без строчных букв
  -d, --no-digits       Без цифр
  -s, --no-special      Без специальных символов
  -c, --count N         Количество паролей
  -o, --output FILE     Сохранить в файл
  --check               Проверить на утечки
  --export              Экспортировать настройки
  --import              Импортировать настройки
  -h, --help            Справка

Примеры:
  python password_generator.py -l 32 -s
  python password_generator.py -c 10 -o passwords.txt
```

## Зависимости

- `customtkinter` — GUI фреймворк
- `pyperclip` — работа с буфером обмена
